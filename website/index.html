DCTimerAI

Android GAN MoYu QiYi

支持姿态跟随与一键重置的智能魔方计时器

GitHub Releases v2.2.4 直接下载 APK

智能连接

  1. 兼容 GAN、MoYu、QiYi 智能魔方，以及部分蓝牙计时器设备。
  2. 支持 MoYu AI / MoYu32 姿态数据读取，整体转动时 3D 魔方实时跟随。
  3. 保留智能魔方转层动画、还原状态识别、成绩记录与统计能力。
  4. 主页面与状态弹窗共用同一套 3D 渲染逻辑，显示更一致。

版本改进

  1. 新增一键姿态重置，可快速校准为白顶绿前的标准视角。
  2. 重建智能魔方 3D 外观与层转表现，主界面和弹窗观感统一。
  3. 保留原项目成熟的计时、成绩、打乱、导入导出与统计功能。
  4. 适合作为个人维护版持续迭代，也便于拆分通用功能回馈上游。
